Edamame

Edamame automates the provisioning and configuration of AWS infrastructure needed to support simulating up to 200,000 concurrent virtual users in a load test.

It comes with a CLI and React GUI that enable developers to easily conduct load tests, visualize metrics relevant for HTTP and WebSocket protocols in near real-time, and stop a test early if the performance of their system under load degrades to an unacceptable threshold.
